Bhavika Art & Craft - MERN E-commerce Website

**Live Website:** https://bhavikaartandcraft.com

Features

-  User Authentication (JWT)
-  Add to Cart with Size-Based Pricing
-  Razorpay Payment Integration
-  Admin Panel for Product Management
-  Fully Responsive Design
-  Deployed on Hostinger


## Tech Stack

### Frontend
- React.js
- Redux Toolkit
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Other Tools
- JWT Authentication
- Razorpay Payment Gateway
